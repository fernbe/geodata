# Madagascar Geodata

Source: The BNGRC (Madagascar National Disaster Management Agency) 

This table adds provinces to the top level of the BNGRC lookup table which uses 4-level administrative divisions.

Administrative Divisions: 
- Level 1 (Provinces)  
- Level 2 (Regions)
- Level 3 (Districts)
- Level 4 (Commune) 
- Level 5 (Fokontany)
